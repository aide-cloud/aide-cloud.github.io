# 5. select

> 1. select 语句类似于 switch 语句，但是 select 会随机执行一个可运行的 case。如果没有 case 可运行，它将阻塞，直到有 case 可运行。
> 2. select 语句中的每个 case 都必须是一个通信操作，要么是发送要么是接收。

## 语法

```go
# 写法一
select {
case <-ch1:
   fmt.Println("A")
case ch2 <- 1:
   fmt.Println("B")
default:
   fmt.Println("C")
}

# 写法二
select {
case <-ch1:
   fmt.Println("A")
case ch2 <- 1:
   fmt.Println("B")
default:
   fmt.Println("C")
}
```

## 常见写法

### 退出信号监听
```go
exit := make(chan bool)
go func() {
    for {
        select {
        case <-exit:
            fmt.Println("exit")
            return
        default:
            fmt.Println("default")
        }
    }
}()
time.Sleep(time.Second * 3)
exit <- true
```

### 超时控制
```go
timeout := make(chan bool, 1)
go func() {
    time.Sleep(time.Second * 2)
    timeout <- true
}()
select {
case <-timeout:
    fmt.Println("timeout")
}
```

### 通道监听
```go
ch := make(chan int, 1)
go func() {
    for {
        select {
        case <-ch:
            fmt.Println("ch")
        default:
            fmt.Println("default")
        }
    }
}()
time.Sleep(time.Second * 3)
ch <- 1
```
