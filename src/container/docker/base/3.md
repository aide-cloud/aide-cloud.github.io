# Dockerfile

## 第一个Dockerfile

- Dockerfile是一个文本文件，用于定义一个镜像的构建过程
```dockerfile
FROM alpine:3.12

RUN echo "Hello World"
```

- 通过`docker build`命令构建镜像

```bash
docker build -t hello-world .
```

- 通过`docker run`命令运行容器

```bash
docker run hello-world
```

## 创建一个go应用的Dockerfile

- 创建一个go应用

```bash
mkdir -p /tmp/go/src/hello-world
cd /tmp/go/src/hello-world
touch main.go
```

- 编写main.go

```go
package main

import (
    "fmt"
    "net/http"
)

func main() {
    http.HandleFunc("/", func(w http.ResponseWriter, r *http.Request) {
        fmt.Fprintf(w, "Hello World")
    })
    http.ListenAndServe(":8080", nil)
}
```

- 创建Dockerfile

```bash
touch Dockerfile
```

- 编写Dockerfile

```dockerfile
FROM golang:1.16-alpine

WORKDIR /app

COPY . .

RUN go build -o main .

EXPOSE 8080

CMD ["./main"]
```

- 通过`docker build`命令构建镜像

```bash
docker build -t my-go-app .
```

- 通过`docker run`命令运行容器

```bash
docker run -p 8080:8080 my-go-app
```

## go应用的Dockerfile优化

- 优化1：使用多阶段构建

```dockerfile
FROM golang:1.16-alpine AS builder

WORKDIR /app

COPY . .

RUN go build -o main .

FROM alpine:3.12

WORKDIR /app

COPY --from=builder /app/main .

EXPOSE 8080

CMD ["./main"]
```

- 优化2：使用scratch镜像

```dockerfile
FROM golang:1.16-alpine AS builder

WORKDIR /app

COPY . .

RUN go build -o main .

FROM scratch

WORKDIR /app

COPY --from=builder /app/main .

EXPOSE 8080

CMD ["./main"]
```
