# 5. 部署一个go项目

## 准备一个go项目

- 创建一个`go`项目

```go
// main.go
// main.go
package main

import (
	"fmt"
	"net/http"
)

func main() {
	http.HandleFunc("/", func(w http.ResponseWriter, r *http.Request) {
		fmt.Fprintf(w, "Hello, %s!", r.URL.Path[1:])
	})
	fmt.Println("Listening on port 8080")
	http.ListenAndServe(":8080", nil)
}
```

- go项目的Dockfile

```dockerfile
# Dockerfile
FROM golang:1.18 AS builder

WORKDIR /app

COPY . .

RUN CGO_ENABLED=0 GOOS=linux GOARCH=amd64 go build -o main .

FROM alpine:latest

EXPOSE 8080

WORKDIR /app

COPY --from=builder /app/ ./

RUN chmod +x main

CMD ["./main"]
```

- 创建一个`docker-compose.yml`文件

```yaml
version: '3.7'

services:
  go:
    build:
      context: .
      dockerfile: Dockerfile
    ports:
      - "8080:8080"
```

- 项目目录结构

```bash
.
├── Dockerfile
├── docker-compose.yml
└── main.go
```

## 部署

- 启动服务

```bash
docker-compose up -d
```

- 查看服务

```bash
docker-compose ps
```

- 查看日志

```bash
docker-compose logs -f
```

- 停止服务

```bash
docker-compose down
```

## 保存镜像   

- 保存镜像

```bash
docker save -o go.tar go:latest
```

- 加载镜像
  - 上传`go.tar`到服务器
  - 在服务器上执行
```bash
docker load -i go.tar
```
